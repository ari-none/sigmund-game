# Sigmund Game
Clone de Simon fait pour un projet de BTS (fait par Arinone et Evan52100).

## Installation
1. Faire un clone du dépot
2. Mettre en place un serveur MariaDB (ou similaire qui peuvent supporter le script `db.sql`)
3. Créer une base de données à partir du script `db.sql`
4. Renommez le fichier `template.env` en juste `.env` et mettez-y vos identifiants à la bonne place
5. Le jeu doit fonctionner :D
6. **[OPTIONNEL]** Créer 150 000 comptes utilisateur en même temps & faire planter la base de données

*Notes : Le jeu peut fonctionner sans mettre en place un serveur MariaDB ; Cela vous empêchera juste d'accéder aux scores en ligne et aux comptes.*
